用 JavaScript 写的一个用来演示 ANTLR 语法分析工具的游戏。limejs, JavaScript, ANTLR, html5

使用的是 ANTLR 3.3， 最新的 3.4 无法编译出可用的 JavasScript 的代码，runtime 用的依旧是 3.4 的。
